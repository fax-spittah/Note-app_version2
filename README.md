# NoteAppVersion2

This is the second and updated version of my previous note application. The difference now is that this time, firebase SDK is used for authentication and storage of other elements. Unlike the previous version, database related tasks are now asynchronous and need to be handled carefully lest fetching, storing and other db related functions will malfunction.
